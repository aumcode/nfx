ExtractMainColors(Bitmap, Int32, Int32, Single)

Extracts three main colors and background color from source image by the following algorithm:

  1. source image color quality reduced down to 256/dwnFactor1 color per each of RGB channel (222=8 base colors used as default)
  2. four areas (three main and one background) with the biggest color frequencies are taken
  3. color frequency analisys is performed in each area which gives area main color (image color quality reduced down to 256/dwnFactor2 color per each of RGB channel)

Background area search is limited to [1-interiorPct, interiorPct] portion of image interior
