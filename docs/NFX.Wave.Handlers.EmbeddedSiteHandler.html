VersionSegmentPrefix

When set indicates the case-insensitive prefix of a path segment that should be ignored by the handler path resolver. Version prefixes are used for attaching a surrogate path "folder" that makes resource differ based on their content. For example when prefix is "@", path '/embedded/img/@767868768768/picture.png' resolves to actual '/embedded/img/picture.png'
